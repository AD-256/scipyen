Running scipyan in Spyder 3

At the moment, scypian can be run in spyder3 as a separate process:

  1. Import scipyan as a project in spyder3:

NOTE This is equivalent to running pict.py from the scypian directory, in a shell.

This procedure starts a completely python3 process, with its own PyQt5 event loop, embedded IPython kernel and namespace (user workspace).

  • Menu "Projects" -> "New Project".
    • In the new project dialog:

      • choose "Existing directory"

      • select the location of scypian, e.g.:

/home/cezar/scipyan

      * select project type: "Empty project" 
                from the drop-down combo box
  1. Open the project (if project already created) by selecting its location, e.g.:

/home/cezar/scypian

  1. Open pict.py in spyder editor

  2. Menu "Run" -> "Run"

NOTE: When running for the first time, the "Run configuration per file" dialog shows up:

  • Choose "Execute in an external system terminal"
  1. On subsequent runs: just select "Run" from the "Run" menu;
    • for other options, select "Configuration per file" from the "Run" menu.
